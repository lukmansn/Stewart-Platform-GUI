# Stewart-Platform-GUI
This is a code for stewart platform user interface 
This user interface provides many features such as:
 - Monitoring part
   a. Servo angle monitor
   b. Output Response
   c. Motion Platform
   d. Real-time data log
 - Controlling part
   a. Independent kontrol panel
   b. Slider kontrol panel
   c. Harmonic motion
 - View camera
   a. eye
   b. 3D view
   c. front view
   d. back view
   e. right view
   f. left view
   g. top view
   h. bottom view
 - Communication
   a. reload
   b. send data
   c. stop send data
   d. switch connect
 - Logging Data
   a. record data
   b. download data

Default condition:
Baudrate = 115200
